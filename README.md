## Description
# Delivery Driver Platform API

This is a backend API for a delivery driver platform that connects scooter owners with trusted and vetted drivers who can rent scooters for a fee. The platform supports JWT authentication for secure access to protected api.

## Features

- User Registration and Authentication (JWT)
- Scooter listing and management
- Driver verification system
- Admin controls for managing users, drivers, and scooters

## Tech Stack

- **NestJS**: A framework for building scalable Node.js applications.
- **Prisma**: An ORM for database access.
- **Passport**: Authentication middleware.
- **JWT (JSON Web Tokens)**: Authentication mechanism.
- **PostgreSQL**: Database (configured via Prisma).
